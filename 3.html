다양한 프로그래밍 언어들

프로그래밍 언어를 선택할 때는 내 컴퓨터나 기기가 어떤 언어를 이해하는지, 내가 만들 프로그램이 어떤 언어와 잘 맞는지 고민해야 합니다.

프로그래밍 언어는 인간 언어와 비슷합니다. 한국어, 영어, 일본어와 같이 나라마다 쓰는 언어가 다르듯이, 프로그래밍 언어도 같은 내용을 저마다 다른 방식으로 사용하지요

하지만, 분명한 차이는 프로그래밍 언어는 인간 언어와 달리 지시한 작업만 특별히 잘한다는 거에요. 아래에서 대표적인 프로그래밍 언어를 살펴볼까요?




스크래치 (Scratch) - 그림을 많이 사용하는 프로그램이나 게임을 만들 때 유용합니다. 누구나 사용하기 쉽습니다.

파이썬 (Python) - 여러 분야에 효과적으로 쓰입니다. 여러분은 혹시 영화 '해리 포터'와 '캐리비안의 해적'을 보셨나요? 이 영화들은 특수 효과가 무척 뛰어난 영화들입니다.
이 영화들의 특수 효과를 만드는데 ILM(Industrial Light & Magic)이라는 프로그램이 사용되었는데요, 바로 파이썬으로 만들어졌습니다.

C와 C++ - 3D 게임과 같이 매우 빠르게 실행하는 프로그램을 만들 때 사용합니다.

자바 (Java) - 안드로이드 스마트폰 게임 및 에플리케이션을 만들 때 사용합니다. 또한 에이컨과 같은 기계들과 결합하여 사용하기도 합니다.

자바스크립트 (JavaScript) - 웹 사이트에서 펼침 메뉴, 텍스트를 바꾸는 기능뿐만 아니라, 온라인 게임처럼 사용자와 상호 작용하는 기능을 만들 때 사용합니다.


자바와 자바스크립트는 이름은 비슷하지만, 전혀 다른 언어랍니다!
